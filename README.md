# トウキ合戦

Quizizz、Kahoot!などのゲームベースの学習プラットフォーム向けの問題ファイル（Excel）を生成する小道具です。

## 技術情報

1. 開発言語：**python3**

2. ライブラリー：

   `エクセル処理` **xlrd** **xlwt**

   ```shell
   pip install xlrd xlwt
   ```

   `データベース` **sqlite3**
